ELEMENTS OF HTML

1.BLOCK LEVEL ELEMENT

Block Level Elements:the tag which occupies whole width and create a new line.

EX:h1-h6,p,div,semantic elements

2.INLINE-BLOCK ELEMENTS

INLINE-BLOCK ELEMENTS:the elements which occupy only require space and does not create a new line and we can apply width,height,margin andpadding properties.

EX:img, iframe,input, button.

3.INLINE ELEMENTS

INLINE ELEMENTS: the elements which occupies only required space and does not create a new line,in css we cannot apply width, height,padding top and bottom and margin top and bottom

EX:span, formatting tags, b, strong, i
